# Mini CAREN Redesign
**The Mini CAREN Redesign** is a project developed to work in the Mini CAREN at the Courage Center. It is intended to replace the old Deflow + OptiTrack system with a Unity + OptiTrack system. 
Class diagrams, Figma frames, and a Wiki is available on the Mini CAREN Confluence page. 


# Installation
1) Install **Unity Version 6000.0.48f1**
2) Add the **Windows Build Support (Mono)** module to your Unity installation
3) Clone or download the GitHub repository
4) In Unity Hub, open the downloaded repository


# Updating the Unity Build
If you are having issues with building the project, make sure to remove any name spaces that use **UnityEditor**.
